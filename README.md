# Module 2 Assignment - Salary Function
# Author: Sodiq Omoniyi

## Overview
This project demonstrates:
- Importing salary data
- Processing data using a Python dictionary
- Creating a function to retrieve employee details
- Implementing error handling
- Exporting employee data to CSV
- Compressing the CSV into a ZIP file
- Using R to unzip and display the employee data

## Files
- Salary_Function.ipynb
- unzip_employee_profile.R
- Employee_Profile.zip

## Running the Python Code
1. Open the Jupyter Notebook.
2. Run all cells.
3. Enter an employee name when prompted.
4. The employee profile CSV and ZIP file will be generated.

## Running the R Code
1. Open RStudio.
2. Run unzip_employee_profile.R.
3. The ZIP file will be extracted and the employee data displayed.
